# React Video Calling App

A simple WebRTC video calling demo built with React + Vite.

## Features

- Start local camera/microphone
- Start a video call in the same tab (local peer loopback)
- View local and remote video panels
- End the call and clean up media tracks

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite and allow camera/microphone access.

## Build

```bash
npm run build
npm run preview
```
